# Semãntica

## Antes do HTML 5

> O site era construído basicamente por divs e ids, portanto não eram semântico.

```html
<!DOCTYPE html>
<html lang="pt-BR">

<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  
  <title>Título</title>
</head>

<body>
  <div id="header"></div>
  
  <div id="main"></div>
  
  <div id="footer"></div>
</body>

</html>
```

<br>

## Depois do HTML 5

> Com o surgimento do HTML 5 veio várias TAGs, que por sua vez cada uma possuia um significado, tornando os sites mais semântico.

```html
<!DOCTYPE html>
<html lang="pt-BR">

<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  
  <title>Título</title>
</head>

<body>
  <header></header>
  
  <main></main>
  
  <footer></footer>
</body>

</html>
```
